## Get Start

Add the following <img/> tag in your 404 page
```html
<img alt="404" 
     src="https://illustatus.herokuapp.com/?title=Oops,%20Page%20not%20found&fill=#4f86ed"/>
```

#### supported parameters

- `fill` - controls the color
- `title` - controls the wording 


and your'll get:

<img alt="404" 
         src="https://illustatus.herokuapp.com/?title=Oops,%20Page%20not%20found&fill=#4f86ed">
